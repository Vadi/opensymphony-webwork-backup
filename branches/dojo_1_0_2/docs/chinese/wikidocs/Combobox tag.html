<ww:combobox />

组合框是由HTML的text和SELECT输入控件组合而成, 提供了组合框的功能. 它可以通过Select控件将文本放入Text中, 也可以直接在Text字段中输入.

本例中, SELECT控件使用id=year属性生成. Counter是一个迭代器(Iterator).

<ww:bean name="'webwork.util.Counter'" id="year">
  <ww:param name="'first'" value="text('firstBirthYear')"/>
  <ww:param name="'last'" value="2000"/>

  <ww:combobox label="'Birth year'" size="6" maxlength="4" name="'birthYear'" list="#year"/>
</ww:bean>

属性
名字 必需 描述
id 否 HTML的id属性
name 是 HTML的name属性
value 否 作为字段值
list 否 列表使用的可迭代数据源
size 否 HTML的size属性
maxlength 否 HTML的maxlength属性
disabled 否 HTML的disabled属性
tabindex 否 HTML的tabindex属性
onkeyup 否 HTML的onkeyup属性
onchange 否 HTML的onchange属性
onclick 否 HTML的onclick属性
cssClass 否 HTML的class属性
cssStyle 否 HTML的style属性
label 否 用于模版中的标签文字
labelposition 否 标签的对齐方式 (left,right,center)
theme 否 使用的主题
template 否 使用的模版名
