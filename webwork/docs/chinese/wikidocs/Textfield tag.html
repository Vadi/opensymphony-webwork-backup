This page last changed on Dec 14, 2004 by casey.

<ww:textfield />

本例中, 绘制了一个文本控件. 它通过调用ActionSupport的getText()方法从ResourceBundle中提取文本标签. 
<ww:textfield label="text('user_name')" name="'user'"/>

属性
名字 必需 描述
id 否 HTML的id属性
name 是 HTML的name属性
size 否 HTML的size属性
maxlength no HTML的maxlength属性
readonly no HTML的readonly属性
disabled 否 HTML的disabled属性
tabindex 否 HTML的tabindex属性
onchange 否 HTML的onchange属性
onclick 否 HTML的onclick属性
label 否 用于模版中的标签文字
labelposition 否 标签的对齐方式 (left,right,center)
required 否 在表单提交时是否是必需字段
cssClass 否 HTML的class属性
cssStyle 否 HTML的style属性
theme 否 使用的主题
template 否 使用的模版名

注意: 本标签的缺省模版是text.vm而不是textfield.vm.
