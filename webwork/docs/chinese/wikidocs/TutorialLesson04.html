第四课: 视图

你可以将多种不同的技术作为视图, 也就是创建用户界面:

4.1 - Java Server Pages

JSP是通常的选择, 因为大部分Java开发者已经熟悉这一技术. 本节假定你已经具备JSP开发经验, 演示如何使用WebWork的特性, 主要是使用WebWork标签.

跳转到 4.1

4.2 - Velocity

Velocity是一个基于Java的模版引擎, 提供了一个简单却强大的模版语言用以替代JSP并允许分离关注点. 本节假定你已经熟悉了Velocity, 并介绍如何应用WebWork特性.

条转到4.2

4.3 - Freemarker

Freemarker是为MVC模式设计的另一个基于Java的模版引擎提供了强大的模版语言用以代替JSP, 但可以通过对JSP标签库的支持保留与JSP的兼容. 本节介绍如何将WebWork和Freemarker结合使用.
