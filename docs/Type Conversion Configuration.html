Type conversion configuration follows the following search order:

  • ActionName-conversion.properties
  • BaseClassName-conversion.properties (all the way up to Object-conversion.properties)
  • InterfaceName-conversion.properties

In these files you specify a name/value pair. The name is the property name you wish to apply a converter to. The value is the custom type converter class that you have build. This class must implement OGNL's TypeConverter interface.
