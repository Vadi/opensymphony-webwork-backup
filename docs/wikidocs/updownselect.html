Important

Note that the listkey and listvalue attribute will default to "key" and "value" respectively only when the list attribute is evaluated to a Map or its decendant. Other thing else, will result in listkey and listvalue to be null and not used.

Description

Create a Select component with buttons to move the elements in the select component up and down. When the containing form is submited, its elements will be submitted in the order they are arranged (top to bottom).

Parameters

An error occurred: http://svn.opensymphony.com/svn/webwork/docs/tags/UpDownSelect.html. The system administrator has been notified.

Examples

<!-- Example 1: simple example -->
<ww:updownselect 
list="#{'england':'England', 'america':'America', 'germany':'Germany'}" 
name="prioritisedFavouriteCountries" 
headerKey="-1" 
headerValue="--- Please Order Them Accordingly ---" 
emptyOption="true" />

<!-- Example 2: more complex example -->
<ww:updownselect 
list="defaultFavouriteCartoonCharacters" 
name="prioritisedFavouriteCartoonCharacters" 
headerKey="-1" 
headerValue="--- Please Order ---" 
emptyOption="true" 
allowMoveUp="true" 
allowMoveDown="true" 
allowSelectAll="true" 
moveUpLabel="Move Up"
moveDownLabel="Move Down" 
selectAllLabel="Select All" />
