<ww:token />

令牌标签用于解决"双击"提交问题. 这需要设置TokenInterceptor和TokenSessionInterceptor. 关于他们的文档参见wiki. ww:token标签仅放置一个包含为一令牌的隐藏元素.

用法示例 

<ww:token />

属性
名字 必需 描述
name 否 令牌名
theme 否 使用的主题
template 否 使用的模版名
