WebWork 2.2.7 Release Notes

Key Changes

There's no major changes in this release. The more significant improvements are probably bumping up WebWork DWR support to DWR 2.x and adding i18n validation parameterizing support.

The remaining changes are pretty trivial, including a few issues backported from Struts2, configurable start up and shut down listener and some trivial bug fixes.

See below for the complete list of changes.

Migration Notes

This 2.2.7 release, is basically a maintaintance release.

It is fully backwords compatible with WebWork 2.2.6 and should be an easy and straight forward drop in replacement for it.

As usual, we'd like to thank both users and contributors for helping out, submitting patches, testing, providing feedback and participating in various discussion related to WebWork.

Once again thanks for using WebWork.
