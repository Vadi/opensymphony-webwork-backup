Else Tag







    
Remarks

    

        
This tag is used to determine if the preceding test was false.

    


    
Details

    

        

          

            Class:
            com.opensymphony.webwork.views.jsp.ElseTag
          

          

            Syntax:
            <ww:else> ... </ww:else>
          

          

            Tag Body:
            Jsp
          

        

    


    
Attributes

    

        
This tag has no attributes.

    


    
Changes from WebWork 1

    

        
Add changes.

    


    
Example

    

        
In this example, the body of the else tag will be evaluated since the test condition of the if tag
        evaluates to false.


<ww:if test="true == false">
   <b>if: Failures</b>
</ww:if>

<ww:else>
   <b>else: Success</b>
</ww:else>


        
Output:


   <b>else: Success</b>

ElseIf Tag







    
Remarks

    

        
This tag is used to determine if a statement is true or false after a previous test.

    


    
Details

    

        

          

            Class:
            com.opensymphony.webwork.views.jsp.ElseIfTag

            Tag Body:
            Jsp
          

        

    


    
Attributes

    

        

          

            Attribute
            Description
            Required
          

          

            test
            This attribute is the conditional expression evaluated by the
                OGNL parser.
                It returns boolean true or false.
            true
          

        

        

         - Inheirited from superclass.

    


    
Changes from WebWork 1

    

        
Add changes.

    


    
Example

    

        
In this example, the body of the elseif tag will be evaluated since its test condition is true and
        the test condition of the if tag evaluates to false.


<ww:if test="true == false">
   <b>if: Failures</b>
</ww:if>

<ww:elseIf test="true == true">
   <b>elseIf: Success</b>
</ww:elseIf>

<ww:else>
   <b>else: Failure</b>
</ww:else>


        
Output:


   <b>elseIf: Success</b>

Param Tag







    
Remarks

    

        
This tag allows you to add parameters to JspTags that implement the ParameterizedTag
        interface. Place one or more param tags within the body of a tag supporting parameters
        and the param tag will add its parameter to its parent.

URL Tag







    
Remarks

    

        
This tag creates an encoded URL using the value specified. If a value is not specified, the URL will point
        to the current page. Parameters can be added to the URL by using the param tag.

    


    
Details

    

        

          

            Class:
            com.opensymphony.webwork.views.jsp.URLTag
          

          

            Syntax:
            <ww:url ... /> [or] <ww:url> ... </ww:url>
          

          

            Tag Body:
            Jsp
          

        

    


    
Attributes

    

        

          

            Attribute
            Description
            Required
          

          

            id  
            If this attribute is set, the encoded URL will be placed in the request scope of the pageContext
                object under the value specified for the 'id' attribute.
            false
          

          

            value
            This attribute specifies the value to encode. If this value is not specified, the
                URL will point to the current page.
            false
          

        

        

         - Inheirited from superclass.

    


    
Changes from WebWork 1

    

        
Add changes.

    


    
Example

    

        
In this example, the param tag is used to add name/value pairs to the specified action. Parameters
            can also be specified using the querystring; i.e., - "hiturl.action?user=john".


<form action="<ww:url value="'hiturl.action'">
      <ww:param name="'user'" value="'john'" />
   </ww:url>" method="POST">
