WebWork and XWork are build countinuously whenever a commit is done. The tests are run and finally a jar file is being bundled up.

The jar files of WebWork and XWork as a result of successful countinuous build could be found at:-

  • Ivy Repository ( http://ivyrep.opensymphony.com/ )
  • Maven 1 ( http://maven.opensymphony.com/ )
  • Maven 2 ( http://maven2.opensymphony.com/ )

This is helpful for users wiling to try out the latest build that might have bug fixes critical to them instead of building from scratch.
