Tag Name: <ww:checkboxlist />

Description

Render a list of checkboxes

Creates a series of checkboxes from a list. Setup is like <ww:select /> or <ww:radio />, but creates checkbox tags.

Examples 

  <ww:checkboxlist name="foo" list="bar"/>
