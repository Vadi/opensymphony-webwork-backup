Webwork troubleshooting

Allrighty, trying to get Webwork2 installed, and this is what I have found so far.

Make sure you include all the dependent libraries. You find these in 'lib/core' from CVS.

You need to place 'xwork.xml' and 'components.xml' in your WEB-INF/classes directory. You can find sample files in 'src/etc/example' and 'src/webapp/WEB-INF' from CVS.

Components are part of Xwork. If you don't have 'components.xml', you get an error. An empty file with the following tags seem to be fine.

<components><component/></components>
