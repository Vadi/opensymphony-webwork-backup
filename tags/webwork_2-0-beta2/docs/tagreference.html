WebWork 2 Tag Reference

This guide will be a reference to all of the JSP tags in WebWork2, including their attributes and some usage examples.

Overview

There are 6 broad categories of JSP tags:

  • Common tags - the most frequently used, basic tags
  • Componentisation tags - foster componentisation within your views
  • Flow control tags - govern the flow of control within the JSP page
  • Iteration tags - iterate over elements and manipulate iterable objects
  • UI tags - generate HTML form fields and controls
  • Internationalisation tags - internationalise your views.

Note: At the moment this tag reference just lists which tags have been completed in WebWork2 and which haven't - it does not give full syntax examples and attribute lists.

Tags which have not been implemented yet are in italics.
