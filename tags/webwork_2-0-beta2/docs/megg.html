Quick boiler-plate WebWork2 project with megg

megg will generate a basic hello world style WebWork2 project for you,with your favourite package name, desired action name, etc. All wrapped up in a lovely little ant script.
  1. Download megg from http://sf.net/projects/megg
  2. Run as shown below.
  3. Copy webwork jar files into project lib directory. (This is detailed by megg after you generate project)
  4. change directory into created project and run ant dist or ant deploy

c:projects>java -jar megg_0_0_4.jar
