WebWork : WebWork Tutorial - Example 3

The sample description

This demo will replace the webwork default template with cutomized one.

The sample include

  • one view files: ex03.vm
  • two template files: checkbox.vm & mytextfield.vm
  • generated html page in portal server

Here is the sample list

ex03.vm 
<html> 
<head> 
<title>WebWork
                        Tutorial - Example 3</title>
<meta name="help-path" content="/help/help3.html"/>
                        
</head> 
<body> 
<H2>Template
                        & Theme Change Example:</H2> 
<br/> More detail please refer to template/components
                        directory.
<p>
#wwcheckbox ("name=checkbox" "label=A checkbox" "fieldValue=checkbox_value" "theme=xhtml/components")
</p> 
<p>
#wwtextfield
                        ("name=textfield" "label=A text field" "template=components/mytextfield.vm")
</p> 
</body>
                        
</html>

checkbox.vm in java class path, this will replace the webwork default checkbox template

template\xhtml\components\checkbox.vm 
<div align="center"> 
 <input type="checkbox" 
              name="$!webwork.htmlEncode($parameters.name)" 
              value="$!webwork.htmlEncode($parameters.fieldValue)" 
       #if ($parameters.nameValue) checked="checked" #end 
       #if ($parameters.disabled == true) disabled="disabled" #end 
       #if ($parameters.tabindex) tabindex="$!webwork.htmlEncode($parameters.tabindex)" #end 
       #if ($parameters.onchange) onchange="$!webwork.htmlEncode($parameters.onchange)" #end 
       #if ($parameters.id) id="$!webwork.htmlEncode($parameters.id)" #end 
       /><br /> 
 $!webwork.htmlEncode($parameters.label)
                            
</div>

mytextfield.vm in java class path, this will replace the webwork default textfield template

mtemplate\xhtml\components\mytextfield.vm generated html page in Portal Server
<div align="center">

       #if ($parameters.size) size="$!webwork.htmlEncode($parameters.size)" #end

       #if ($parameters.nameValue) value="$!webwork.htmlEncode($parameters.nameValue)" #end
