WebWork 2 : 部署WebWork注意事项
本页由plightbo最后修改于2004年5月24日.

WebWork可以在大部分应用服务器上运行而没有任何问题. 尽管如此, 你可能还是需要做不多的改动以便在你的环境下运行.

WebLogic 6.1

一个子项目已被加入WebWork的CVS仓库中, 它极大的简化了使WebWork在BEA Weblogic Server 6.1下运行所需的的修改工作. 相关文档也包含在其中.
该子项目位于目录"misc"中.

SunONE 7.0

需要为WebWork分配许可:
grant {
	permission java.security.AllPermission;
};

或更明确地,

  • 分配对java.util.PropertyPermission的写许可.
  • 增加java.lang.reflect.ReflectPermission "suppressAccessChecks"
  • OgnlInvoke许可
grant {
	permission java.util.PropertyPermission "*", "read, write";
	permission java.lang.reflect.ReflectPermission "suppressAccessChecks";
	permission ognl.OgnlInvokePermission "*";
};
