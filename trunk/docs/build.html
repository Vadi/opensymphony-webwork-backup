WebWork2 Documentation

Building WebWork 2

Download packaged distribution

WebWork2 is downloadable from https://webwork.dev.java.net/servlets/ProjectDocumentList

or… Checkout from CVS

For those who want to get the CVS copy, checkout the webwork module from CVS.

cvs -d :pserver:anoncvs@cvs.dev.java.net:/cvs login

followed by

cvs -d :pserver:anoncvs@cvs.dev.java.net:/cvs checkout webwork

Build WebWork 2 JAR

Jump into the webwork directory and run ant clean jar. This will build WebWork 2 JAR file.

What do I do now?

You should have a JAR in webwork2/build/webwork-2.0.jar that you can play with!

Build the WebWork2 Example App

Even better, you can build an example WAR that should deploy on any Servlet 2.3 container that can help you get a feeling for the features in WebWork2 even quicker. To build this, just run ant example-war and deploy webwork2/build/webwork-example.war in your favorite app server.
