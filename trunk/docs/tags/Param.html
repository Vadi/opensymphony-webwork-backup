Tag Name: <ww:param />

Description

Parametrize other tags

This tag can be used to parameterize other tags, who implement the ParametricTag interface declared here.

The include tag and bean tag are examples of such tags.

The inner classes, Parametric and UnnamedParametric, are implemented by tags. They indicate that a particular Tag may have embedded params. For example, if we were wanted to use the ComponentTag and wanted to provide custom params to assist with the rendering, we could declare something like

  • name (String) - the name of the parameter
  • value (Object) - the value of the parameter

Examples 

  <ui:component> <ui:param name="key" value="[0]"/> <ui:param name="value" value="[1]"/> <ui:param name="context" value="[2]"/> </ui:component>

where the key will be the identifier and the value the result of an OGNL expression run against the current OgnlValueStack.

Attributes

Name

Required

Default

Type

Description
name true   String Name of Parameter to set
value false The value of evaluating provided name against stack Object/String Value expression for Parameter to set
id false   Object/String id for referencing element. For UI and form tags it will be used as HTML id attribute
