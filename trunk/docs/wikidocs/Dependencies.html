WebWork has only a few required dependenies but has many optional dependencies. You can determine what these dependencies are by looking in the docs/dependencies directory of the distribution, or by clicking here. The only required dependencies are those in the default configuration. If you plan to use the UI tags, you will also need the jars in the freemarker configuration (unless you plan to write all your UI templates from scratch in a different language, such as Velocity or JSP, which is not recommended).
