WebWork : WebWork Tutorial - Example 2

The sample description

This demo will demo how to use webwork standard UI component, and xwork action in portal application.

The sample include

  • two view files: ex02.vm & ex02Success.vm
  • one xwork file: xwork.xml
  • one java action file: FormProcessingAction.java
  • one validation file: FormProcessingAction-validation.xml
  • generated html page in portal server

				<interceptor-ref name="conversionError" />
			</interceptor-stack>
		</interceptors>
