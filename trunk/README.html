Thank you for downloading WebWork2! The WebWork2 goal is to create the best Model-2 MVC web application framework supporting advanced web application development paradigms such as component based development and code reuse.

Key Features

  • Built on XWork, but customized to be specifically tailored for web application development
  • Custom web-specific Xwork interceptors
  • Flexible configuration extended from XWork with web-specific configuration parameters
  • Multiple web-based views, including custom JSP taglibs, Velocity support with pre-built macros, XSLT views, and JasperReports
