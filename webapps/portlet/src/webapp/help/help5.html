WebWork : WebWork SiteMesh Menu Tutorial - Example 5

The sample description

This sample will demo how to use sitemesh as decorator to decorate html page as menu

The sample include

  • one view files: menu.vm
  • one sitemesh configuration file: decorators.xml
  • one sitemesh decoration file: menuDecorator.vm
  • generated html page in portal server

Here is the sample list

menu.vm, the main content file will be rendered
<html> 
<head> 
 <title>WebWork
                        SiteMesh Menu Tutorial - Example 5</title> 
</head> 
<body>
                        
 <h1>Custom Menu Example:</h1>
                        
 <h2>WebWork SiteMesh Menu Tutorial</h2>
 <table>

                        <tbody>
 <tr><td>
 <div class="tabletitle">decorated
                        content:</div>
 <div class="preformatted ">
 The main content
                        displayed here!
 </div>
 </td></tr>

                        </tbody>
 </table>
</body>
                        
</html>

sitemesh decorator configuration file: decorators.xml in WEB-INF
<decorators defaultdir="/decorators">

                        <!-- No decorator -->

                        <decorator name="none">

                        <!-- in case some appservers decide they /can/ decorate 404 pages -->

                        <url-pattern>/fourohfour.action</url-pattern>
 <url-pattern>/styles/*</url-pattern>
 <url-pattern>/index.vm</url-pattern>
 <url-pattern>/help/*</url-pattern>
 </decorator>

                        <decorator name="menu" page="menuDecorator.vm">
 <pattern>/menu.vm</pattern>
 </decorator>

                        <decorator name="edit" page="editDecorator.vm">
 <pattern>/edit/*</pattern>

                        </decorator>
 <decorator name="index" page="indexDecorator.vm">

                        <pattern>/index.vm</pattern>

                        </decorator>
 <decorator name="main" page="mainDecorator.vm">

                        <pattern>*</pattern>
 </decorator>

                        <decorator name="printable" page="printable.vm" />

                        <decorator name="menupanel" page="menupanel.vm" />
</decorators>
