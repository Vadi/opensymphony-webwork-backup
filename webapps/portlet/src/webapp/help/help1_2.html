WebWork : WebWork JSP Demo 1-2

The sample description

This sample demo how to use JSP as display script in wwportlet framework.

The sample include

  • two view files: ex01.jsp & ex01Result.jsp
  • one xwork file: xwork.xml
  • one java action file: HelloAction.java

Here is the sample list

hello.jsp
<%@ taglib uri="webwork" prefix="ww"%>
<html>
<head><title>Please input your information</title></head>
<body>
<p>Welcome to Use <ww:property value="now"/>
</p>
<ww:form action="'/jsp_hello.action'" method="'post'">
 <ww:textfield label="User Name:" name="userName" value="%{userName}" />
 <ww:submit name="Submit" value="Submit"/>
</ww:form>
<br/>
</body>
</html>

helloResult.jsp
<%@ taglib uri="webwork" prefix="ww"%>
<html>
<head>
<title><ww:text name="heading" /></title>
<!-- meta name="help-path" content="/help/help1_2.html"/ --> 
</head>
<body>

 <ww:textfield label="User Name" name="userName"/><br>

</body>
</html>
